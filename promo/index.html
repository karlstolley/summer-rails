Learn Rails

Take a deep dive into open-source web development powered by Ruby on Rails. This production-intensive course covers the practice and applied theory of full-stack web application development. You will develop a thorough understanding of model-view-controller architecture (MVC), which will be coupled with bespoke database-driven application design, executed through object-relational mapping (ORM). Course projects afford the opportunity for you to leverage MVC architecture to develop and interact with open data APIs expressed in serialized formats such as JSON. You will also engage in mobile-first, responsive interface design using standards-compliant forms of HTML, CSS, and the DOM and a suite of a modular preprocessors including Haml, Sass, and CoffeeScript.

The Hard Sell: Take This Class

Whether or not you have professional ambitions in web development, there are numerous higher-level principles and practices that this class introduces:

  • The professional necessity of tracking the development of active open-source projects
  • Exposure to highly opinionated and uncompromising MVC architecture
  • Full-stack development that closely knits the front- and back-end logic and execution common to MVC architecture
  • Mapping user requests and HTTP verbs onto system internals and methods, which leverage the capability of modern clients and browsers
  • Intensive work with consoles and loggers to up your debugging game outside of clients and browsers
  • Test-driven development (TDD) to ensure code quality and internal consistency, and to streamline code reviews
  • Object-relational mapping (ORM) that abstacts away databases and reconciles the set-theory behind databases with the structures of classes and methods that drive object-oriented programming
  • Continuous integration (CI) as part of robust test-driven development and automated deployment

Week-by-Week Overview

  1. Week 1: Ruby Fundamentals

    • Ruby syntax, structures, and style
    • Ruby standard libraries and core classes
    • Navigating and referencing Ruby documentation
    • Open-source culture and practices

  2. Week 2: Rails Fundamentals

    • MVC architecture
    • Agile development techniques
    • The rails command
    • Rails environments (development, testing, production)
    • Gemfiles, Bundler, and semantic versioning
    • The Rails community history and culture

  3. Week 3: The Structure of Applications

    • Models and migrations
    • Object-relational mapping (ORM)
    • Bespoke data structures and the primacy of models
    • Routing and controllers
    • Functional and unit tests; fixtures

  4. Week 4: Views and Assets

    • Views and layouts in Rails
    • The asset pipeline in Rails
    • Haml, Sass, and CoffeeScript
    • Helper methods

  5. Week 5: Sessions and Users

    • Session variables and persistence
    • OAuth-based user authentication
    • File uploads
    • Integration testing

  6. Week 6: Deployment and Continuous Integration

    • Automated deployment with Capistrano
    • Virtualizing test deployments
    • Testing setup for continuous integration

Book List

  • Pine, Christopher. Learn to Program. 2nd ed. Pragmatic Bookshelf. $18.50 (eBook). ISBN 9781934356364

    Required. Although titled Learn to Program, Pine’s book is one of the best introductions to the Ruby language. It’s also a solid refresher on general programming concepts and object-orientedness.

  • Ruby, Sam, Dave Thomas, and David Heinemeier Hansson. Agile Web Development with Rails 5. Pragmatic Bookshelf. $28.00 (eBook). ISBN 9781680501711

    Required. The industry-standard book for learning Rails development. Buy the electronic edition from Prag to continue to receive updates (as with all Pragmatic Bookshelf titles).

  • Swicegood, Travis. Pragmatic Version Control Using Git. Pragmatic Bookshelf. $25.00 (eBook). ISBN 9781934356159

    Not required, unless you have little to no experience with command-line Git. Recommended read ahead of class.

  • Thomas, Dave, with Chad Fowler and Andy Hunt. Programming Ruby 1.9 & 2.0. 4th ed. Pragmatic Bookshelf. $28.00 (eBook). ISBN 9781937785499

    Not required, but no serious Rubyist should be without this exhaustive and authoritative reference to the Ruby programming language.

My policy for assigning books: purchase only the edition indicated; the total retail pricetag for the entire course should be less than $100 (this one is $99.50 on the high end, assuming you opt for the electronic editions and the two optional titles); and each book should be worthy of a place on your bookshelf or electronic device of choice long after the class has ended.

Projects

There will be two major projects in this class. An individually developed prototype alpha app, and a group developed beta app culled from the individual alpha apps. Additionally, graduate students will be expected to more deeply investigate one of the core Rails framework’s modules (e.g., Active Record, Action Cable, Active Support) and serve as in-house consultants on them.

Required Tech Setup

  • A Unix-like operating system (Linux, BSD, macOS, etc.). Virtualization is fine, but Windows is not.
  • A plain-text editor that highlights Ruby syntax (there is no widely-used IDE for Ruby). If you’re an Atom user, you’ll want the Ruby language package for Atom.
  • A command-line Git installation and GitHub account.
  • The Node.js JavaScript runtime.
  • A Ruby environment, such as rbenv (recommended) or RVM.

Full setup details will be emailed to registered students ahead of the first class meeting.
